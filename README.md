# 🇩🇪 Ultimate Flashcards — Learn German

A premium, high-performance Progressive Web App (PWA) designed for mastering German vocabulary using a sophisticated spaced-repetition system (SRS).

🚀 **[Open the App](https://vibeuna.github.io/GermanFlashcards/)**

---

## ✨ Features

- **3D Flashcard Engine**: Interactive study mode with smooth CSS-driven 3D flip animations.
- **Spaced Repetition (SRS)**: Advanced scheduling with 6 customizable review grades (Wrong, Correct, Semi-easy, Easy, 1 Month, 2 Months).
- **OCR Scanning**: Use your camera to scan physical text (books, signs, menus) and instantly convert them into flashcards with auto-translation via Tesseract.js.
- **Glassmorphism Design**: A premium, modern aesthetic with frosted glass elements and dynamic animated gradients.
- **Privacy First**: All data is stored locally in your browser (`localStorage`). No servers, no tracking.
- **PWA Support**: Install it on your phone or desktop for a native-app feel and full offline support.
- **Data Portability**: Easily import or export your vocabulary collections via CSV files.

---

## 🛠️ Technical Stack

- **Frontend**: Vanilla HTML5, CSS3 (Custom Design System), and ES6+ JavaScript.
- **OCR**: [Tesseract.js](https://tesseract.projectnaptha.com/) for on-device text recognition.
- **PWA**: Custom Service Worker for assets caching and Offline storage.
- **Styling**: Modern CSS variables, glassmorphism filters, and Inter typography.

---

## 💻 Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/vibeuna/GermanFlashcards.git
   ```
2. Serve the directory using any local server (e.g., Live Server in VS Code, or `npx serve .`).
3. Open `localhost` in your browser.
